  1. Get XCode with command line tools and MacPorts. See earlier notes from when I needed these things for an unrelated project.
  2. Install unixODBC with sudo port install unixodbc +universal.
  3. Get the right Teradata ODBC driver. As of this writing that is 14.10, not 15.00, even though the latter looks like it would be more recent and it shows up at the top of the list in bigger type. 15.00 does not work, as documented here.
